# mjbots pi3hat #

This contains designs, firmware, and client side libraries for the
mjbots pi3hat.  It is a daughterboard for Raspberry Pi (tm) boards and
similar to provide:

* 4x 5Mbps CAN-FD ports
* 1x 125kbps CAN port
* 1kHz IMU with attitude reference
